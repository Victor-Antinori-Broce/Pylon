### Notas Técnicas

#### Sistema de Registro de Componentes
Reemplaza el PluginRegistry de Vue con una implementación nativa de Svelte:

```typescript
// Registrar un widget
componentRegistry.registerWidget({
  id: "mi-widget",
  slot: "dashboard:top",
  component: MiWidget,
  order: 10,
  module: "mi-modulo",
  props: { titulo: "Hola" }
});

// Usar en template
<WidgetSlot name="dashboard:top" wrapper="card" />
```

#### Features Implementadas
- **Widgets**: Registro en slots nombrados (dashboard:top, dashboard:sidebar, etc.)
- **Rutas Dinámicas**: Soporte para rutas inyectadas por módulos
- **Bloques**: Sistema de bloques para el page builder
- **Acciones**: Toolbar y context menu actions
- **Filtrado por Módulos**: Solo muestra widgets de módulos habilitados

#### Renderizado Dinámico
Usa `<svelte:component>` para montar componentes dinámicamente:

```svelte
<script>
  let MiComponente = $state(null);
  
  // Cargar dinámicamente
  onMount(async () => {
    MiComponente = (await import('./MiComponente.svelte')).default;
  });
</script>

{#if MiComponente}
  <svelte:component this={MiComponente} {...props} />
{/if}
```

## Sistema de Renderizado Dinámico

### Uso Básico

```svelte
<script>
  import { componentRegistry } from "./lib/registry.svelte";
  import MiWidget from "./MiWidget.svelte";
  
  // Registrar widget
  componentRegistry.registerWidget({
    id: "mi-widget",
    slot: "dashboard:top",
    component: MiWidget,
    order: 10,
    module: "mi-modulo",
    props: { titulo: "Hola Mundo" }
  });
</script>

<!-- Renderizar slot -->
<WidgetSlot name="dashboard:top" wrapper="card" />
```

### Componente Dinámico

```svelte
<script>
  import DynamicComponent from "./lib/DynamicComponent.svelte";
  import MiComponente from "./MiComponente.svelte";
  
  let props = { mensaje: "Hola" };
</script>

<DynamicComponent component={MiComponente} {props} />
```

### Comparación Vue vs Svelte

| Vue | Svelte |
|-----|--------|
| `<component :is="MiComponente" />` | `<svelte:component this={MiComponente} />` |
| `PluginRegistry.registerWidget()` | `componentRegistry.registerWidget()` |
| `<slot name="empty" />` | `<slot name="empty" />` (mismo) |
| `v-bind="props"` | `{...props}` |
| `ref()` para estado | `$state()` |

## APIs de Extensibilidad

Apps externas pueden extender el CMS registrando widgets:

```typescript
// En una app externa
declare const gremius: typeof import('./lib/registry').componentRegistry;

gremius.registerWidget({
  id: "mi-extension-widget",
  slot: "dashboard:stats",
  component: MiWidget,
  order: 5,
  module: "mi-extension"
});
```
